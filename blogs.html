Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM by   1.getElementById()  2.getElementsByClassName()  3.getElementsByTagName()  4.querySelector()  5.querySelectorAll()   and  6.getElementsByName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Using innerHTML we 'obtain' or 'set' the HTML tags and contents that ar present in the desired Tag.
Using innerHTML we 'obtain' or 'set' the visible text content of an element, excluding any hidden elements.
Using textContent we 'obtain' or 'set' the text content of an element, with plain text, ignoring any CSS style.

Question-3: What is event delegation in the DOM?

Event delegation helps us to do our work with less effort. Instead of eventListning multiple elements, we access the parent element and then access the child elements with more efficiency.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event not only triggers itself but also its parent element. The event bubbles up from the target to its root which is the main HTML tag.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element by using the document.createElement method. It creates a new element/HTML tag.
We can add the created elements to the DOM by appendChild() , insertBefore() & insertAdjacentHTML() methods.
We can use the removeChild() or remove() method to remove an element from the JS DOM.
